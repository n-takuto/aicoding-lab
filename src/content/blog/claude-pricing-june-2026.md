2026年6月15日、Anthropicが Claude の料金体系を大きく変える予定でした。

「対話利用」と「自動化利用（Agent SDK）」を別枠に分離するという内容で、発表時は「実質値上げでは？」とXで話題になりました。

※ Agent SDK とは、Claude をプログラムから自動実行するための仕組みです。`claude -p`（Claude Code をパイプラインやスクリプトで動かすオプション）も同じ扱いになる予定でした。

ところが、**この変更は現在一時停止中です（2026年6月16日時点）。**

Anthropic 公式ヘルプには以下の記載があります。

> "We're pausing the changes to Claude Agent SDK usage described below."
> "Claude Agent SDK, claude -p, and third-party app usage still draw from your subscription's usage limits."

つまり、今のところ何も変わっていません。この記事では、**発表された変更の内容**と**なぜ停止になったのか**を整理します。

## 発表されていた変更内容

発表では、6月15日から以下のように利用が2つの枠に分かれる予定でした。

| 利用の種類 | 変更前 | 変更後（予定） |
|-----------|--------|--------------|
| ターミナルでの Claude Code 操作 | サブスクリプション上限 | 変わらず |
| Claudeチャット（Web・デスクトップ） | サブスクリプション上限 | 変わらず |
| **Agent SDK / `claude -p`** | サブスクリプション上限 | **月次クレジット（別枠）** |
| **GitHub Actions 等の自動実行** | サブスクリプション上限 | **月次クレジット（別枠）** |

## プランごとの月次クレジット（予定）

自動化系の利用に対して、以下のクレジットが毎月付与される予定でした。

| プラン | 月額 | 月次クレジット（予定） |
|--------|------|----------------------|
| Pro | $20 | $20 相当 |
| Max 5x | $100 | $100 相当 |
| Max 20x | $200 | $200 相当 |

クレジットが尽きると自動化系の処理は停止し、翌月への繰り越しもなし、という内容でした。

## なぜ一時停止になったのか

Anthropic は停止の理由を詳しく説明していませんが、Fable 5 の停止騒動（Anthropic の新モデルが米政府の輸出規制指令で突然停止した件、[詳細はこちら](/blog/claude-fable-5-suspended/)）が同時期に重なったことも影響している可能性があります。

いずれにせよ、公式が「一時停止」と明言しているので、2026年6月16日時点では Agent SDK を使っても従来通りサブスクリプション上限から消費されます。

## 今どうなってる？（2026年7月4日追記）

[Claude Help Center の公式ページ](https://support.claude.com/en/articles/15036540-use-the-claude-agent-sdk-with-your-claude-plan)を改めて確認したところ、**7月4日時点でも一時停止のまま**でした。

- Agent SDK の別クレジット化：**一時停止中**（変更なし）
- 現在の挙動：**変更前と同じ**（Agent SDK もサブスクリプション上限から消費）
- Anthropicは「サブスクリプションでの使い方をよりよくサポートできるよう計画を練り直している」とし、再実施の前には事前告知すると表明
- 再実施のタイミング：**引き続き未定**

## まとめ

- 6月15日に「対話利用」と「自動化利用」を別枠に分ける変更が予定されていた
- 直前で一時停止となり、**7月4日時点でも変更前のまま**
- Anthropicは計画を練り直し中で、再実施のタイミングは未発表

変更が再開される際には改めて告知があるはずなので、Anthropicの公式アナウンスを確認しておくといいでしょう。この記事も続報が出たら更新します。
