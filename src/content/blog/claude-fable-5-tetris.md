2026年7月7日、今日は **Claude Fable 5 がサブスクリプションの範囲内で使える最終日** です。

「せっかくなら何か試したい」と思い、Claude Code のモデルを Fable 5 に切り替えて、**テトリスを丸投げ**してみました。<br>
結果：**1分30秒で、修正ゼロの一発完成でした。**

以前「[Claude Code にマインスイーパを丸投げしたら2分で動いた話](/blog/claude-code-minesweeper/)」という記事を書きましたが、今回はその Fable 5 版です。

## Fable 5とは？7月7日で何が変わる？

Claude Fable 5 は、Anthropic の最上位モデルです。6月に輸出規制騒動で一時停止して、7月1日に復活した経緯は[こちらの記事](/blog/claude-sonnet-5-and-fable-5-return/)にまとめています。

復活後の提供条件が、今日を境に変わります。

| 期間 | Fable 5 の使い方 |
|------|----------------|
| **〜2026年7月7日** | Pro / Max / Team / 一部Enterpriseプランで、週次利用上限の50%まで追加料金なし |
| **2026年7月8日〜** | 使用量クレジット制（サブスクとは別に課金） |

つまり「サブスク料金だけで気軽に試せるのは今日まで」。駆け込みで使ってみた、というわけです。

## 投げたお願いは実質「テトリス」だけ

Claude Code デスクトップ版で `/model` から Fable 5 に切り替えて、お願いした内容はほぼこれだけです。

```
じゃあテトリス
```

その前に「何か作るならどんなお題がいい？」と候補を出してもらっていたので、文脈込みとはいえ、実質 **「テトリス」の一言** です。要件も仕様も何も伝えていません。

> 💡 マインスイーパの時は箇条書きで要件を渡しましたが、今回はあえて丸投げ度を上げてみました。

## 結果：1分30秒で一発完成

時刻を記録しながら見ていたので、正確な時間が残っています。

| 時刻 | 出来事 |
|------|--------|
| 21:07:33 | 計測開始、コードを書き始める |
| 21:09:03 | HTML/CSS/JavaScript 1ファイル完結のテトリスが完成 |

**コードを書き上げるまで1分30秒**。しかもブラウザで開いたら、**修正なしでそのまま動きました**。

![起動直後のタイトル画面](/images/tetris/01-title.png)

### 頼んでないのに本格仕様だった

驚いたのは中身です。「とりあえず動くテトリス」ではなく、ゲームとしてちゃんと遊べる仕様が最初から入っていました。

| 実装されていた機能 | どんなもの？ |
|------------------|------------|
| **SRS回転法則（壁蹴り）** | 壁際でも回転が引っかからない、現代テトリス標準の回転ルール |
| **7種1巡の出現順** | 7種類のミノ（落ちてくるブロックのこと）が偏りなく出る、公式ルール準拠の抽選方式 |
| **ゴーストピース** | 落下位置が半透明で表示される「影」 |
| **ホールド** | 今のミノを取っておいて後で使える機能 |
| **NEXT表示（3つ先まで）** | 次に来るミノの予告 |
| **レベルアップで加速** | 10ライン消すごとに落下速度が上がる |

これ、全部こちらから頼んでいません。「テトリスと言えばこの仕様だよね」という水準を勝手に満たしてきた感じです。

![プレイ中の画面。左のHOLDと右のNEXTがちゃんと機能している](/images/tetris/02-playing.png)

## 前回のマインスイーパとどう違った？

2026年5月に書いたマインスイーパ記事と比べると、違いが分かりやすいです。

| | マインスイーパ（2026年5月） | テトリス（今回・Fable 5） |
|---|--------------------------|------------------------|
| 渡した指示 | 要件を箇条書きで渡した | 実質「テトリス」の一言 |
| 完成まで | 2分16秒 | **1分30秒** |
| 途中の修正 | Claude が自分でバグを2回修正 | **修正ゼロ** |
| ゲームの複雑さ | クリック判定中心 | 回転法則・衝突判定・ライン消去など段違い |

実装の難易度はテトリスの方が明らかに上です（回転の壁蹴り処理だけでもかなり複雑）。それが、より雑な指示で、より短時間で、修正なしで出てくる。モデルの世代差を体感できた瞬間でした。

## 「動いた」の中身をちゃんと検証

「動いた！」だけだと本当に遊べるか分からないので、動作確認もしました。

**① 自動テストで一通りチェック**<br>
Claude Code のプレビュー機能で、移動・回転・ハードドロップ（ミノを一気に下まで落とす操作）・スコア加算・ホールド・ポーズ・ゲームオーバー・リスタートまで、キー入力をシミュレートして確認。全部正常でした。

**② 最後は自分で遊んで確認**<br>
唯一自動でテストしづらかったのが「ライン消去」（ランダムに落ちてくるミノで狙って1列揃えるのが難しい）。ここは私が実際にプレイして、ちゃんとラインが消える・スコアが入るところまで確かめました。検証と言いつつ普通に遊んでました。

![ゲームオーバー画面。スコア・積み上がったブロックまでちゃんと表示される](/images/tetris/03-gameover.png)

## スマホ対応にして、実際に遊べます

1分30秒で出てきたのはキーボード操作のPC版でした。その後「スマホでも遊べるように」と追加でお願いして、タッチ操作ボタンを付けてもらっています（この作業は1分30秒には含まれていません）。

![スマホ表示。画面下に移動・回転・ホールド・ハードドロップのボタンが並ぶ](/images/tetris/04-mobile.png)

完成したテトリスは、このブログに置いてあります。PCでもスマホでも遊べます。

## まとめ

- 2026年7月7日は Fable 5 がサブスク範囲内で使える最終日だった（7月8日からは使用量クレジット制）
- 「テトリス」の一言で、**1分30秒・修正ゼロ**の一発完成
- SRS回転法則・ホールド・ゴーストなど、頼んでいない本格仕様まで入っていた
- マインスイーパ（2分16秒・自己修正2回）と比べて、難しいものを速く正確に作れるようになっていた

Fable 5 は明日からクレジット制になりますが、Claude Code の開発リーダーは「容量が確保でき次第、サブスクの標準機能として戻したい」とX（旧Twitter）で表明しています（2026年7月確認）。また気軽に使える日が来たら、次はもっと大きなものを丸投げしてみたいです。
