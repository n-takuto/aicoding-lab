以前「[Claude Fable 5が3日で停止した理由を調べた](/blog/claude-fable-5-suspended/)」という記事で、「復旧したらまた試してみます」と書いていたのですが、まさかのタイミングでその日が来ました。

2026年6月30日、Anthropicから**新モデル「Claude Sonnet 5」**が発表され、同じ日に**停止していたFable 5の輸出規制も解除**。今回は、この2つの動きを公式情報ベースでまとめます。

## Claude Sonnet 5とは

Claude Sonnet 5は、2026年6月30日にリリースされた新モデルです。Anthropicは「**最もエージェント指向のSonnetモデル**」と位置づけていて、ブラウザやターミナルなどのツールを使った自律実行に強みを持ちます。

**主な特徴：**

| 項目 | 内容 |
|------|------|
| リリース日 | 2026年6月30日 |
| 利用できる場所 | Claude.ai（全プラン）、Claude Code、Claude Platform |
| APIでのモデル名 | `claude-sonnet-5` |
| 性能 | 前世代のSonnet 4.6比で、推論・ツール利用・コーディング・知識作業が大幅改善 |
| コンピュータ操作タスク | Opus 4.8に迫る精度を、より低コストで達成（VentureBeat報道） |

前世代のSonnet 4.6からの伸び幅が大きく、「Opus 4.8に近い性能を、もっと安く」というのが今回のアピールポイントのようです。

## 料金（期間限定の導入価格あり）

料金は2段階になっています。

| 期間 | 入力（100万トークン） | 出力（100万トークン） |
|------|---------------------|---------------------|
| **〜2026年8月31日**（導入価格） | $2 | $10 |
| **2026年9月1日〜**（通常価格） | $3 | $15 |

導入価格は2026年7月1日時点で残り2ヶ月ほどの期間限定なので、API経由で試すなら早めがお得かもしれません。

## トークナイザーが更新されている点に注意

Anthropicの発表によると、Sonnet 5では**トークナイザー（文章をトークンに区切る仕組み）が更新**されています。同じ文章を入力しても、以前より**約1.0〜1.35倍のトークン数**になる可能性があるとのこと。

これはOpus 4.7のときにも同様の変更がありました。API経由で使っていて「あれ、思ったよりトークン数が増えた?」と感じたら、このトークナイザー更新が原因かもしれません。

## Claude Fable 5がついに復活

そしてもう1つの動きが、**Fable 5の輸出規制解除**です。

以前の記事で書いた通り、Fable 5は2026年6月9日にリリースされたものの、わずか3日後の6月12日に米政府の輸出規制指令で全世界一斉停止していました。

きっかけは、Amazonの研究者がFable 5の安全対策を回避する方法を発見し、ソフトウェアの脆弱性を特定できる可能性を示したレポートだったそうです。これを受けて米政府から出されたのが**輸出規制指令**（安全保障上の理由で、特定の国・地域へのモデル提供を制限する米政府の措置）でした。Anthropicは国籍をリアルタイムで確認する手段がなかったため、**全ユーザーへのアクセスを一時中断**する対応を取っていました。

Anthropicの公式発表によると、**2026年6月30日付で輸出規制が解除**され、**7月1日（水）から**Claude Platform、Claude.ai、Claude Code、Claude Coworkで再びグローバルに利用できるようになる予定です。

なお、この発表にタイムゾーンの明記はありませんでした。Anthropicは米国（太平洋時間）を拠点にしているため、日本時間だと反映まで半日〜1日ほどのズレが生じる可能性があります。実際、私のClaude Codeでも2026年7月1日時点ではまだFable 5が使えませんでした。すぐ使えなくても焦らず、少し時間を置いて確認してみてください。

**復活後の条件：**

| 項目 | 内容 |
|------|------|
| 利用開始日 | 2026年7月1日（Anthropic発表・タイムゾーン記載なし） |
| 利用できる場所 | Claude Platform、Claude.ai、Claude Code、Claude Cowork |
| 料金の優遇 | Pro / Max / Team / 一部のEnterpriseプランは7月7日まで、週次使用制限の50%まで無料 |
| 7月7日以降 | クレジット制で提供 |
| その他プラットフォーム | AWS、Google Cloud、Microsoft Foundryでの再開は今後の予定 |

停止から復活まで、ちょうど3週間弱でした。

## 2つのニュースが重なった今週

整理すると、2026年6月30日〜7月1日のAnthropicはこんな流れでした。

- **6月30日**：Claude Sonnet 5 リリース
- **6月30日**：Fable 5の輸出規制が解除
- **7月1日**：Fable 5がグローバルで利用再開（Anthropic発表ベース。日本時間では反映にズレがある可能性）

新モデルの登場と、話題になっていたモデルの復活が同じタイミングで重なったので、Anthropic関連のニュースを追っている身としてはかなり動きの大きい1週間でした。

## まとめ

- **Claude Sonnet 5**：2026年6月30日リリース。前世代より大幅に改善、Opus 4.8に迫る性能を低コストで。導入価格は2026年8月31日まで
- **Claude Fable 5**：輸出規制が解除され、2026年7月1日からグローバルで利用再開予定（発表にタイムゾーン記載なし、日本では反映が遅れる場合あり）。Pro/Max/Team/一部Enterpriseは7月7日まで一部無料
- トークナイザー更新でトークン数が変わる可能性があるので、API利用者は注意
